Embrace the path of Earth


Addendum

Process:

The game is structured using a finite state machine approach, where each state represents a stage in the game's narrative.

The gameData object holds information about each state, including the text to display, available choices, consequences of those choices, and associated images.

Two main functions, startGame() and updatePage(), handle game initialization and updating the display, respectively.

Creative Choices:

The game theme revolves around elemental paths, Earth and Sky, offering players distinct choices and consequences.

Each state is designed to immerse players in a vivid environment, featuring descriptive text and accompanying images to enhance the storytelling experience.

Endings are tailored to reflect the player's choices, offering a sense of accomplishment and closure.

Challenges Faced:

Dynamic Content Rendering: Dynamically updating the page content based on player choices posed a challenge, particularly managing transitions between states seamlessly.

Event Handling:Ensuring smooth event handling for player choices and updating the game state accordingly required careful attention to event listeners and callback functions.

Resource Management: Handling image resources efficiently, including loading and displaying images without causing performance issues, was crucial.

How Challenges Were Overcome:

Modular Design: Breaking down the game logic into modular functions aided in better organization and debugging, mitigating issues related to event handling and state transitions.

Testing and Debugging: Rigorous testing and debugging were conducted at each stage of development to identify and address issues promptly.

Optimization: Optimizing resource loading and updating strategies helped maintain smooth gameplay without sacrificing performance.

Insights Gained:

User Experience: Understanding the importance of immersive storytelling and engaging visuals in enhancing the user experience.

State Management: Appreciating the significance of effective state management in interactive applications, facilitating dynamic content updates and player progression.

Feedback Loop: Recognising the value of iterative development and user feedback in refining and improving the game experience over time.
